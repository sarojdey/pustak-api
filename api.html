Welcome to the Book API

This API provides endpoints to retrieve information about books.

Endpoints:

  • GET /api/book - Get all books
  • GET /api/book/:id - Get book by ID
  • GET /api/book/genre/:genre - Get books by genre
  • GET /api/book/author/:author - Get books by author
  • GET /api/book/search/:query - Search for books by title, author, or genre

Usage:

To retrieve books, make a GET request to the corresponding endpoint.

For example:

  • To get all books: /api/book
  • To get a book by ID: /api/book/123
  • To get books by genre (replace spaces with hyphens): /api/book/genre/science-fiction
  • To get books by author (replace spaces with hyphens): /api/book/author/j-k-rowling
  • To search for books (replace spaces with hyphens): /api/book/search/harry-potter
